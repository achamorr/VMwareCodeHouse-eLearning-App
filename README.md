## To Run

Git clone this project to your computer

CD to directory where this project is

Download dependencies:
    `npm install`

Run: 
    `node start`


## To Test

On command line:
`curl http://localhost:3000`

In browser go to http://localhost:3000/images/kitten.jfif

## To kill
On CL on Windows: 
`taskkill -F -IM node.exe`
